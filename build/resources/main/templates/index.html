Applicant endpoints

  • GET /api/applicant/{username}

    Params
    {username} String

  • POST /api/applicant/new

    Params

    RequestBody
    Application/JSON;UTF-8

    { "username": "demo123456784", "password": "A123abc456+", "passwordConfirmation": "A123abc456+", "email": "cv@mm.com" }

  • PUT /api/applicant/{username}

    Params
    {username} String

    RequestBody
    Application/JSON;UTF-8

    { "username": "demo123456784", "address1": "MX", "address2": ".", "country": ". ", "email": "cv@mm.com", "lastname": "Test22", "names": "Demo test", "state": "aa", "zipcode": "." }

  • DELETE /api/applicant/{username}

    Params
    {username} String

  • PUT /api/applicant/{username}/update-pwd

    Params
    {username} String

    RequestBody
    Application/JSON;UTF-8

    { "usernameChangePwdForm": "demo123456784", "currentPassword": "A123abc456+", "password": "A12132d43+++1", "passwordConfirmation": "A12132d43+++" }

Applicant Skills endpoints

  • GET /api/skills/{username}

    Params
    {username} String
    {page} Optional Integer
    {rows} Optional Integer

  • POST /api/skills/{username}/new

    Params

    RequestBody
    Application/JSON;UTF-8

    { "name": "Objective C", "proficiency": 4 }

  • PUT /api/skills/{id}

    Params
    {id} Long

    RequestBody
    Application/JSON;UTF-8

    { "name": "F#", "proficiency": 3 }

  • DELETE /api/skills/{id}

    Params
    {id} Long
